# План разработки магазина обуви

## 1. Проверка и подготовка

- Убедиться, что `myshop` и `cart` находятся в `INSTALLED_APPS`.
- Проверить настройки `Shop/settings.py` для статических и медиа файлов.
- Подключить маршруты приложений в `Shop/urls.py`.
- Создать файл `cart/urls.py`.

## 2. Бэкенд

- Настроить модели `Category` и `Product`.
- Убедиться, что `Product` содержит поля:
  - `category`, `name`, `slug`, `description`, `price`, `image`, `is_available`, `created_at`.
- Реализовать админку для управления товарами.
- Написать вьюхи:
  - `product_list`
  - `product_detail`
  - `category_products`
  - `cart_detail`
  - `cart_add`
  - `cart_remove`
- Проверить работу корзины через `Cart(request)`.

## 3. Фронтенд

- Создать `templates/base.html`.
- Создать шаблоны:
  - `products/product_list.html`
  - `products/product_detail.html`
  - `cart/cart_detail.html`
- Сделать навигацию по категориям и ссылку на корзину.
- Связать форму добавления товара с `cart_add`.
- Оформить стили в `static/home.css`.

## 4. Тестирование

- Проверить, что список товаров отображается.
- Проверить страницы категории и товара.
- Добавить товар в корзину и удалить его.
- Убедиться, что фото товара загружается через `MEDIA_URL`.
- Проверить работу админки.

## 5. Дальше

- Реализовать поиск и фильтрацию.
- Добавить оформление заказа.
- Добавить авторизацию и личный кабинет.
